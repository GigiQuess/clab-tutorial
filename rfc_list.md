# 📘 TCP/IP の主要 RFC 一覧（必須コア → 拡張 → 運用）

## 🟥 **1. コアプロトコル（TCP/IP の中心）**

| 分類 | RFC | 内容 |
|------|------|------|
| **IP（Internet Protocol）** | **RFC 791** | IPv4 の基本仕様。パケット構造、フラグメント、ルーティングなど。 |
| **ICMP** | **RFC 792** | Ping やエラーメッセージの仕組みを定義。 |
| **TCP（最新仕様）** | **RFC 9293** | TCP の最新統合仕様。RFC 793 を完全に置き換える。 |
| **TCP（旧仕様）** | **RFC 793** | 1981 年のオリジナル TCP 仕様。現在は RFC 9293 により廃止。 |
| **UDP** | **RFC 768** | 軽量なコネクションレス通信の仕様。 |

---

## 🟦 **2. アドレッシング・ARP・基盤技術**

| 分類 | RFC | 内容 |
|------|------|------|
| **ARP** | **RFC 826** | IPv4 アドレスを MAC アドレスに解決する仕組み。 |
| **Assigned Numbers** | **RFC 820** | ポート番号やプロトコル番号の割り当て。 |
| **Ethernet** | IEEE 802 系だが、TCP/IP と密接に関連。 |

---

## 🟩 **3. ルーティング関連 RFC**

| 分類 | RFC | 内容 |
|------|------|------|
| **IGP（内部ルーティング）** | RFC 2328 | OSPFv2 |
| **EGP（外部ルーティング）** | RFC 4271 | BGP-4（インターネットの基盤） |
| **CIDR** | RFC 4632 | クラスレスアドレッシングの仕様 |

---

## 🟨 **4. TCP の重要拡張 RFC（実装者向け）**

RFC 9293 は TCP の最新統合仕様ですが、周辺の拡張も重要です。

| 分類 | RFC | 内容 |
|------|------|------|
| **ECN** | RFC 3168 | 混雑通知（Explicit Congestion Notification） |
| **SACK** | RFC 2018 | Selective ACK による高速再送 |
| **Fast Recovery** | RFC 5681 | TCP 輻輳制御の標準 |
| **RTO 計算** | RFC 6298 | 再送タイマの計算方法 |
| **セキュリティ強化** | RFC 5961 | RST 攻撃対策（RFC 9293 により一部更新） |

---

## 🟧 **5. アプリケーション層（TCP/IP 上で動く主要プロトコル）**

| プロトコル | RFC | 内容 |
|------------|------|------|
| **DNS** | RFC 1034 / 1035 | 名前解決の基本仕様 |
| **HTTP/1.1** | RFC 9112 | Web の基盤 |
| **SMTP** | RFC 5321 | メール送信 |
| **FTP** | RFC 765 | ファイル転送（古典的） |
| **Telnet** | RFC 854 | 旧来のリモートログイン（現在は SSH が主流） |

---

## 🟫 **6. 運用ガイドライン・ベストプラクティス**

| 分類 | RFC | 内容 |
|------|------|------|
| **Host Requirements** | RFC 1122 / 1123 | TCP/IP 実装者向けの必須要件 |
| **Internet Architecture** | RFC 1958 | インターネット設計哲学 |
| **Robustness Principle** | RFC 793 / 9293 に含まれる | “Be conservative in what you send, liberal in what you accept.” |
